# Game Performance & Marketing Efficiency Analysis

## Project Overview
This project analyzes how marketing spend impacts user growth and revenue performance across different games.

---

## Analysis Focus

### 1. Performance Analysis
- Total users per game  
- Revenue performance comparison  
- Growth potential across games  

### 2. Marketing Efficiency
- Ad spend vs user acquisition  
- Cost per user (CPU)  
- Revenue vs marketing investment  

### 3. Business Insights
- Which games scale efficiently  
- Which games overspend on acquisition  
- Where marketing can be optimized
